## Project Topic
In this project, a Python program was developed that automatically generates
an HTML web page using information provided by the user.

## Project Purpose
The purpose of this project is to enable the student to generate a static web
page by using string operations and file writing (File Write) capabilities in
the Python programming language.

## Technologies Used
- Python
- HTML
- CSS

## Working Principle of the Program
1. When the program is executed, the user is asked to enter their name, courses,
   and a short biography.
2. The entered courses are separated by commas and converted into a list.
3. The collected information is embedded into an HTML structure.
4. The generated HTML code is written into an `index.html` file.
5. After completion, the program displays a success message to the user.

## How to Run the Program
1. Python must be installed on the computer.
2. Navigate to the `src` directory.
3. Run the following command:
   src/index_olusturucu.py
4. Open the generated `index.html` file in a web browser.
